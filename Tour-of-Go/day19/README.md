# Day 19: Functions and Exercises

Welcome to Day 19 of my Go learning journey! 🚀

In this folder, you'll find my hands-on experiments and exercises with Go functions. Here, I explore how to define, call, and use functions in various ways, along with some practical challenges to deepen my understanding.

## What's Inside?
- `functions.go`: My notes and code samples on Go function syntax and usage.
- `exercise.go`: Fun exercises I solved to practice writing and using functions.

### How to Run
Open your terminal, navigate to this directory, and run:

```sh
go run <filename.go>
```

Replace `<filename.go>` with either `functions.go` or `exercise.go` to see the code in action.
