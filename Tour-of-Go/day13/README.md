# Day 13: Pointers and Structs in Go

This folder contains examples and explanations for two important Go concepts:

## Pointers
- Demonstrates how pointers hold memory addresses of values.
- Shows how to use `&` to get the address and `*` to dereference pointers.
- Example: Modifying values via pointers.

## Structs
- Introduces structs as collections of fields.
- Shows how to declare, initialize, and access struct fields.
- Includes examples of pointers to structs and modifying struct fields.

Files:
- `main.go`: Entry point, runs struct pointer examples.
- `pointers.go`: Pointer concepts and usage.
- `structs.go`: Struct declaration, usage, and pointer interaction.
