# Day 09: Go If Statements

This folder demonstrates Go's `if` statement features:

- **Basic if**: Simple condition check
- **If with short statement**: Declare/init variable in if
- **If-else**: Branching with variable scope

## Files
- `if.go`: Example functions
- `main.go`: Runs the examples

## Run
```sh
go run main.go if.go
```

Output may vary due to random numbers.
